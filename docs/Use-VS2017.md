# Use-VS2017

## SYNOPSIS
Load the build enviroment for Visul Studio 2017

## SYNTAX

```
Use-VS2017 [<CommonParameters>]
```

## DESCRIPTION
Load the build enviroment for Visul Studio 2017 using the Visual Studio 2017 development settings.

## EXAMPLES

### EXAMPLE 1
